# Docs
<br>


## Requirements

* link
* apache2
* php 8


## Install

### via composer
```
composer require global i74ifa/vaid
```

### **Manual**

clone this repo

extract file 


``` 
ln -snf /path/to/vaid/vaid /usr/local/bin

```

<div style="color: red;">You need absolute path</div>

## use

### link virtual host 

```
sudo vaid link 
```

### unlink virtual host 
```
sudo vaid unlink 
```
